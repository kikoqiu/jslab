Advanced JSLab Examples

This document provides more detailed examples of advanced functionalities available in JSLab, building upon the basic concepts introduced in the "Getting Started" guide. Many of these examples demonstrate JSLab's capabilities for high-precision numerical computing and dynamic data visualization.

Remember that all these examples, and more, can be loaded and run interactively within the JSLab application via the "Help" menu.

Working with Large Numbers

BigInt Operations

JSLab fully supports JavaScript's native BigInt type, allowing you to perform calculations with arbitrary-precision integers, exceeding the limits of standard JavaScript numbers. This is particularly useful for cryptographic operations, scientific computing, or any scenario requiring extremely large integer values.

echo('Js buildin bigint
');
let res=1n;
for(let i=2n;i<=50n;++i){
  res*=i;
}
latex(math.parse('50!'),'=',res); // Calculates 50 factorial and displays it using LaTeX.

High-Precision Floating-Point with bfjs

For scenarios demanding higher precision than standard JavaScript Number types, JSLab integrates libbf.js, accessible via the global bfjs object. This library provides arbitrary-precision floating-point arithmetic, essential for sensitive scientific and engineering computations.

  • bfjs.sin(x): Calculates the sine of x.
  • bfjs.PI(): Returns the value of Pi to high precision.
  • bfjs.sqrt(x): Calculates the square root of x.
  • bfjs.acos(x): Calculates the arc cosine of x.
  • value.toFixed(precision, roundingMode): Formats a bfjs number to a specified precision.
// Demonstrate high-precision sine and PI
let value1=bfjs.sin(2 * bfjs.PI());
latex(math.parse('sin(2\pi)'),'=',value1.toFixed(10,150));
echo('Expected result for sin(2π) is 0.');
echo();

// Verify a trigonometric identity with high precision
let value2=(bfjs.sin(bfjs.PI().div(3)).minus(bfjs.sqrt(3).div(2)));
latex(math.parse('sin(\pi/3)-\sqrt{3}/2'),'=',value2.toFixed(10,150));
echo('Expected result for sin(π/3) - sqrt(3)/2 is 0.');
echo();

// Calculate 2 * arccos(0) to high precision (should be PI)
let value3=bfjs.acos(0).times(2);
latex(math.parse('2 \arccos(0)'),'=',value3.toFixed(10,150));
echo('Expected result for 2 * arccos(0) is PI.');
//big float constants
//set decimal precision to 2000 digits
bfjs.push_decimal_precision(2000);
echo(bfjs.PI().toFixed(10,1999))
bfjs.pop_precision();
//calc definite integration with romberg's method
info={}
bfjs.helper.romberg(x=>4/(1+x*x),0,1,1e-30,1e-30,info);
echo(info);

Dynamic and Advanced Plotting

Plotly Integration

JSLab leverages the powerful Plotly.js library for creating interactive, publication-quality graphs. While the plot() function provides a simplified interface for common 2D plots, you can utilize the full power of Plotly.js for highly customized visualizations. Refer to the Plotly.js JavaScript Reference for extensive options.

Generating Multi-Series Plots Dynamically

You can programmatically generate complex plots with multiple data series. The example below shows how to plot several curves derived from a mathematical function over a range of parameters.

let res=[];
for(let a=1.5;a<4;a+=0.5){
  // Generate x values using range()
  x=range(0,10,0.1);
  // Calculate y values based on 'a' and 'x'
  y=x.map(x_val=>x_val**a/a**x_val);
  // Push both x and y arrays for plotting
  res.push(x);
  res.push(y);
}
// Plot all generated series
plot(...res); // Uses the spread operator to pass multiple x,y pairs to plot()

Exploring More Interactive Examples

For more advanced features, including precise numerical calculations with BigInt and Big Float (bfjs), 3D plotting, and advanced mathematical operations, explore the interactive examples directly within the JSLab application:

  • Go to the "Help" menu in the JSLab application.
  • Select "Load General Usage" to load examples from help.json.
  • Select "Load Plotting Usage" to load examples from helpplot.json.

These will load cells directly into your JSLab environment, allowing you to experiment and learn interactively.
