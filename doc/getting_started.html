Getting Started with JSLab

Welcome to JSLab! This is an interactive JavaScript environment designed for numerical computing, data exploration, and powerful visualizations directly in your browser. With JSLab, you can easily perform complex calculations, generate data, and create dynamic plots.

All the client codes run in a web worker, and the content of `document.body` is redirected to the output area.

Here's a quick guide to some fundamental functionalities:

Printing Output: echo()

The echo() function is your primary tool for displaying text and variable values in the output area of a cell. It accepts multiple arguments, which it prints formatted with HTML line breaks.

echo('Hello JSLab!');
let myNumber = 42;
echo('The answer is', myNumber);
echo(`Current date: ${new Date().toLocaleDateString()}`);

Output HTML content: document.body

The document.body is delegated to the output area with linkedom lib. After the code is executed, the the content of `document.body` is appended to the output area.

echoHTML('Hello JSLab!');
let div=document.createElement(`div`);
document.body.append(div);

//enable animation, when you want to flash the html content before the execution finished.
box.startAnimation();
for(let i=0;i<5;++i){
  div.innerHTML=""+new Date();
  await delay(1000);
}

Working with Math: latex()

JSLab integrates powerful mathematical capabilities through math.js and can render expressions beautifully using LaTeX. Use the latex() function to display mathematical formulas.

// Display a simple equation
latex('E = mc^2');

// Display an expression parsed by math.js
latex(math.parse('f(x) = sin(x) + x^2'));

// Display the result of a derivative (requires 'math' object)
// For this to work, ensure 'math' object is available in your scope (it usually is).
// latex(math.derivative('x^2', 'x')); // This would typically be how you'd use it if math.js was fully exposed for symbolic.
// Let's use a simple string example for getting started.
latex('\\int_0^1 x^2 dx = \\frac{1}{3}');

Generating Sequences: range()

The range() function allows you to quickly generate arrays of numbers, which is incredibly useful for data manipulation and plotting.

  • a: start value (inclusive).
  • b: end value (exclusive).
  • step: (Optional) increment value. Defaults to 1.
  • mapper: (Optional) a function to transform each generated value.
// Generate numbers from 0 to 4
let numbers = range(0, 5); // Result: [0, 1, 2, 3, 4]
echo('Generated numbers:', numbers);

// Generate even numbers from 0 to 8
let evenNumbers = range(0, 10, 2); // Result: [0, 2, 4, 6, 8]
echo('Even numbers:', evenNumbers);

// Generate squares of numbers from 0 to 3
let squares = range(0, 4, i => i * i); // Result: [0, 1, 4, 9]
echo('Squares:', squares);

Visualizing Data: plot()

JSLab provides the plot() function as a simplified interface to the powerful Plotly.js library, enabling you to create interactive 2D line charts and scatter plots effortlessly.

// Plot a simple sine wave
let x_values = range(0, 10, 0.1);
let y_values = x_values.map(x => Math.sin(x));
plot(x_values, y_values, { labels: ['Sine Wave'], layout: { title: 'Basic Sine Wave' } });

// Plot multiple lines
let x_data = range(0, 10);
let y1_data = x_data.map(val => val);
let y2_data = x_data.map(val => val * 2);
plot(x_data, y1_data, x_data, y2_data, { labels: ['Line y=x', 'Line y=2x'], layout: { title: 'Two Lines' } });
